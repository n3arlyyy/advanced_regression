Цены на жильё – продвинутые методы регрессии. Проект посвящен предсказанию цен на жилье при помощи продвинутых методов регрессии. Строится модель, которая предсказывает цены на жилье на основе различных параметров. Используются алгоритмы очистки данных, объединения и сравнения моделей, а также методы визуализации данных для отображения зависимостей данных. Обучение и тестирование модели проводится на отдельных наборах данных. Проводится оценка точности предсказаний.
